# Security Policy

## Supported Versions

Security fixes target the latest published version of `flashpod`.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability. Email the
maintainer listed in `package.json` with:

- A short description of the issue.
- Steps to reproduce it.
- Any affected versions or commits you know about.

`flashpod` generates Python bridge code and can call deployment tooling, so
reports involving command execution, generated code, credentials, or endpoint
requests are especially useful.
